## 🔍 Overview

<img align="right" alt="Security GIF" width="340" src="https://media.giphy.com/media/L1R1tvI9svkIWwpVYr/giphy.gif"/>

**CyberRecon** is a next-generation, **AI-Powered Automated Cyber Reconnaissance and Vulnerability Scanning Platform**, developed as the **Final Year BCA Group Project** at *Banwari Lal Jindal Suiwala College, Tosham (Bhiwani)* under **Chaudhary Bansi Lal University, Bhiwani** — Session **2023–2026**.

In the modern digital landscape, cybersecurity has become one of the most critical domains. Manual reconnaissance and vulnerability assessment are **time-consuming, error-prone, and require deep technical expertise**. CyberRecon bridges this gap by providing:

- ✅ A **unified browser-based interface** integrating **13+ powerful security tools**
- ✅ **Google Gemini AI** for human-readable, actionable intelligence reports
- ✅ **Firebase Realtime Database** for persistent scan history and analytics
- ✅ **Automated PDF report generation** with professional formatting
- ✅ **Full Recon Mode** — complete 360° reconnaissance in a single click

## 🛡️ 13 Scanning Modules

<div align="center">

```
┌─────────────────────────────────────────────────────────────────┐
│              CYBERRECON — RECONNAISSANCE ARSENAL                │
│                      13 Active Modules                          │
└─────────────────────────────────────────────────────────────────┘
```

| # | Module | Layer | Description |
|:--:|:---|:--:|:---|
| 01 | 🌐 **Ping Test** | Network | ICMP latency check — RTT, packet loss, host reachability |
| 02 | 🛰️ **GeoIP Satellite Mode** | Network | IP → Country, City, ISP, ASN & GPS Coordinates |
| 03 | 🔐 **SSL/TLS Security Scanner** | Web | Certificates — issuer, expiry, TLS version, cipher suite |
| 04 | 🧠 **Tech Stack Detector** | Web | Identifies server technology, CMS, JS frameworks |
| 05 | 🔑 **Hidden Admin Path Finder** | Web | Scans for common admin panels & hidden endpoints |
| 06 | 📋 **WHOIS Lookup** | DNS | Registrar, creation/expiry date, owner info |
| 07 | 🔎 **DNS Records Deep Look** | DNS | All record types — A, MX, NS, TXT, CNAME |
| 08 | 🗺️ **Traceroute Geo-Trace** | Network | TCP traceroute with Firewall Bypass Mode — maps each hop |
| 09 | 📡 **HTTP Headers Analysis** | Web | All response headers — highlights missing security headers |
| 10 | 🔗 **Page Link Extractor** | Web | Crawls target, extracts internal & external hyperlinks |
| 11 | 🧮 **Subnet Calculator** | Network | Network class, broadcast address, host ranges from CIDR |
| 12 | 🗄️ **Nmap Deep Scan** | Network | Full SYN scan + OS & version detection (top 1000 ports) |
| 13 | 💉 **SQL Injection Scanner** | Application | Multi-phase automated SQLi vulnerability detection |

</div>

           │                               │
           ▼                               ▼
┌─────────────────────┐       ┌────────────────────────┐
│    TOOLS LAYER      │       │    AI PROCESSING       │
│  /backend/tools/    │       │   ai_analyst.py        │
│  13 Python Modules  │       │   Google Gemini API    │
└─────────────────────┘       └────────────────────────┘
           │                               │
           ▼                               ▼
┌─────────────────────┐       ┌────────────────────────┐

CyberRecon follows a **three-tier client-server architecture** with an additional **AI processing layer** and **cloud-based data persistence layer** — built for scalability and modular extensibility.

## ⚙️ Installation & Setup

### Step 1 — Prerequisites

```bash
sudo apt update && sudo apt install -y python3 python3-pip nmap tcptraceroute git
```

### Step 2 — Clone & Install

```bash
# Clone the repository
git clone https://github.com/CyberNiteshHub/CYBER-RECON.git
cd CYBER-RECON

# Install Python dependencies
pip3 install -r requirements.txt
```

**`requirements.txt` includes:**
```
flask
requests
beautifulsoup4
dnspython
reportlab
python-nmap
urllib3
google-generativeai
python-dotenv
```

### Step 3 — Configure API Keys

```bash
cp env.example .env
# Edit .env and set your keys
```

```env
GEMINI_API_KEY=your_gemini_api_key_here
FIREBASE_DATABASE_URL=https://your-project.firebaseio.com
```

> 🔑 Get **Gemini API Key** → [Google AI Studio](https://ai.google.dev/)  
> ☁️ Get **Firebase URL** → [Firebase Console](https://console.firebase.google.com/)

### Step 4 — Launch

```bash
# Linux / Kali Linux
chmod +x run_linux.sh && sudo ./run_linux.sh

# Windows
run_windows.bat
```

### Step 5 — Access

```
🌐  Open your browser → http://localhost:5000
```

## 🚀 Usage

```
1. 🌐  Open       →  http://localhost:5000
2. 🎯  Target     →  Enter domain, URL, or IP address
3. 🔧  Module     →  Select from 13 tools (or choose Full Recon)
4. ▶️   Scan       →  Click RUN — watch real-time terminal output
5. 🤖  AI Report  →  Click "Generate AI Report" for Cortex analysis
6. 📄  Download   →  Save CONFIDENTIAL PDF or open Dashboard
```

**Supported Target Formats:**
```
example.com             ← Domain
http://example.com      ← URL with protocol
192.168.1.1             ← IP Address
192.168.1.0/24          ← IP Range / CIDR notation
```
