# 🕵️ CyberRecon - OSINT Toolkit

A professional, local GUI-based Information Gathering tool for Ethical Hacking and Reconnaissance.

## ✨ Features
- **One-Click Run:** No manual setup required.
- **Tools:** Ping, Traceroute, DNS, Whois, GeoIP, Nmap, Subnet, etc.
- **Reporting:** Generates professional PDF reports (Single Tool or Full Scan).
- **Interface:** Dark Cyberpunk/Hacker theme.

## 🚀 How to Run

### Windows
1. Double-click `run_windows.bat`
2. Wait for the browser to open automatically.
    
### Linux / Kali
1. Open Terminal.
2. Run: `chmod +x run_linux.sh`
3. Run: `./run_linux.sh`
